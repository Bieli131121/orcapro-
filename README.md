# OrcaPro v7

## Rodar localmente
```bash
npm install
npm run dev
```
Acesse: http://localhost:5173

## Hospedar na Vercel (grátis)
1. Crie conta em vercel.com
2. Instale: npm install -g vercel
3. Na pasta do projeto: vercel
4. Siga os passos — em 2 minutos está no ar!

## Hospedar na Netlify (grátis)
1. npm run build
2. Arraste a pasta `dist/` em app.netlify.com/drop
